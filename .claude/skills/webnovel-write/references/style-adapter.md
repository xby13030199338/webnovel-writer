## 定位
Step 2B 专用提示词，将粗稿改写为网文风格。

**与 Step 4 (polish-guide) 的职责边界**:
- **Step 2B (本步骤)**: 纯风格转换 —— 句式改写、抽象→具体、网文口感、去 AI 味
- **Step 4**: 问题修复 —— 基于审查报告修复 critical/high 问题 + AI痕迹检测

> 注意：Step 4 不会重复执行 Step 2B 的句式转换，两步各有侧重。

## 输入
- 粗稿正文（Step 2A 输出）
- 本章类型（战斗/对话/过渡/突破）
- 钩子要求（类型 + 强度）

## 输出
- 网文化正文（字数 ±10%）
- 改写日志（记录主要调整）

## 禁改项（红线）
- ❌ 剧情走向
- ❌ 事件顺序
- ❌ 角色行为结果
- ❌ 设定/能力描述
- ❌ 伏笔内容

## 必改项
- ✅ 开头120字必须有冲突/风险/强情绪
- ✅ 结尾80-150字设置钩子
- ✅ 长句（>40字）拆分
- ✅ 抽象判断 → 动作/反应/代价
- ✅ 删除“总结式旁白”和“解释段”

## 网文增强规则
- ✅ 章首300字内明确“本章目标 + 当前阻力”（避免空转）
- ✅ 每600-900字给一次“微兑现”（信息兑现/关系兑现/能力兑现）
- ✅ 每章至少1处“可量化变化”（地位/资源/关系/风险任一项）
- ✅ 章末钩子优先使用“选择钩/危机钩”，避免纯口号收尾
- ✅ 对话保持“意图冲突”，减少无效寒暄

## 去 AI 味：核心规则（5条）

> 来源：Humanizer-zh（基于 Wikipedia:Signs of AI writing）

1. **删除填充短语** — 去除开场白和强调性拐杖词
2. **打破公式结构** — 避免二元对比、戏剧性分段、修辞性设置
3. **变化节奏** — 混合句子长度；两项优于三项；段落结尾要多样化
4. **信任读者** — 直接陈述事实，跳过软化、辩解和手把手引导
5. **删除金句** — 如果听起来像可引用的语句，重写它

## 去 AI 味：注入个性与灵魂

缺乏灵魂的写作迹象（即使技术上"干净"）：
- 每个句子长度和结构都相同
- 没有观点，只有中立报道
- 不承认不确定性或复杂感受
- 没有幽默、没有锋芒、没有个性

注入方法：
- **有观点**：不要只是报告事实，要对它们做出反应
- **变化节奏**：短促有力的句子。然后是需要时间慢慢展开的长句
- **承认复杂性**："这令人印象深刻但也有点不安"胜过"这令人印象深刻"
- **允许一些混乱**：完美的结构感觉像算法
- **对感受要具体**：不是"这令人担忧"，而是具体描述

## 去 AI 味：24种模式（Humanizer-zh）

### 内容模式

**1. 过度强调意义/遗产/趋势**
警示词：标志着、见证了、是……的体现/证明、至关重要的、关键转折点、不断演变的格局、不可磨灭的印记
→ 删掉意义宣告，改写为具体事实

**2. 宣传和广告式语言**
警示词：充满活力的、丰富的（比喻）、深刻的、开创性的、著名的、令人叹为观止的、迷人的
→ 用具体细节替代形容词堆砌

**3. 以 -ing 结尾的肤浅分析**
警示词：突出/强调/彰显……、确保……、反映/象征……、培养/促进……
→ AI 在句末加现在分词短语制造虚假深度，直接删除

**4. 模糊归因和含糊措辞**
警示词：专家认为、观察者指出、一些批评者认为、多个来源显示
→ 改为具体来源，或直接删除

**5. 提纲式"挑战与未来展望"**
警示词：尽管……面临若干挑战、尽管存在这些挑战、未来展望
→ 改为具体事件描述

### 语言和语法模式

**6. 过度使用的 AI 词汇**
高频词：此外、至关重要、深入探讨、强调、持久的、增强、培养、突出（动词）、复杂性、格局（抽象名词）、关键性的、展示、织锦（抽象名词）、宝贵的、充满活力的
→ 直接替换为普通词汇或删除

**7. 系动词回避**
警示词：作为/代表/标志着/充当 [一个]、拥有/设有/提供 [一个]
→ 改回"是"

**8. 否定式排比过度使用**
警示词："不仅……而且……"、"这不仅仅是关于……，而是……"
→ 直接陈述核心意思

**9. 三段式法则过度使用**
→ 两项优于三项；四项优于三项；打破对称

**10. 刻意换词（同义词循环）**
→ 同一人物/事物用同一词称呼，不要为了"避免重复"而换词

**11. 虚假范围**
警示词："从 X 到 Y"但 X 和 Y 不在有意义的尺度上
→ 直接列举具体内容

### 风格模式

**12. 破折号过度使用**
→ 改为逗号或句号；每段最多一个破折号

**13. 粗体过度使用**
→ 正文中删除粗体；仅标题/关键术语首次出现时可用

**14. 内联标题垂直列表**
→ 改为流畅段落叙述

**15. 表情符号**
→ 全部删除

### 交流模式和填充词

**16. 协作交流痕迹**
警示词：希望这对您有帮助、当然！、一定！、请告诉我
→ 全部删除

**17. 填充短语**
- "为了实现这一目标" → 删除或改为"为此"
- "值得注意的是数据显示" → "数据显示"
- "在这个时间点" → "现在"
- "系统具有处理的能力" → "系统可以处理"

**18. 过度限定**
警示词：可以潜在地、可能被认为、某种程度上可能
→ 删掉多余限定词，保留一个

**19. 通用积极结论**
警示词：未来看起来光明、激动人心的时代即将到来、继续追求卓越
→ 改为具体的下一步行动或悬念

## 去 AI 味：句式与节奏（网文专项）

- 主要用短句，偶尔插入中长句制造变化
- 单句段落制造停顿和强调
- 碎片化短语增加紧迫感（"我愣住了，然后掏出手机"）
- 逆向逻辑（"我苦笑——所有项目都是我的……"）
- 高紧张时刻省略主语（"浑身是汗，我冲向会议室"）
- 重复关键短语强化情绪（"好吧。他们想单干？让他们去吧。"）
- 用问题打断叙述（"他们只是……不工作了？"）
- 避免连续三句相同句式，至少改一处为短句爆点
- 场景/时间转换用数字标记（"1"）

## 去 AI 味：口语化与个性（网文专项）

- 用日常俚语替代术语
- 加入流行语、地区性用语或角色专属昵称
- 语气词（"吧"、"啊"）增强自然对话感
- 对话中加入犹豫（"等等……这不可能对吧……"）
- 通过粗犷的对话加剧冲突

## 去 AI 味：情感与细节（网文专项）

- 快速生动的动作表现情感（"我的手在颤抖"），而非直接说情绪
- 精确数字增加真实感（"老板递给我40万"）
- 独立的情感转折点单独成段（"我爆发了。"）
- 建立对比（人物处境/态度/命运的反差）

## 分题材风格加权
- **玄幻/修仙/高武**：动作与结果比重更高，术语解释后置。
- **都市/直播/电竞**：信息节奏更快，强化“反馈-反应-反制”三连。
- **言情/替身/狗血**：情绪弧线前置，关键场景必须有关系位移。
- **悬疑/规则怪谈/克苏鲁**：线索投放要可回收，恐惧来自规则而非堆词。

## 行文风格加权

当 `state.json → project.writing_style = “fanqie_shuangwen”` 时，在上述规则基础上叠加：
- **黄金三章**（第1-3章）：300字内核心冲突，500字内点明金手指，1000字内首个爽点
- **先虐后爽**：每章情绪弧线必须是 压抑→反击→打脸→收获
- **零注水**：删除一切与推进剧情无关的段落，背景信息一句话带过
- **悬念连环**：每个爽点结束前埋下下一个期待点
- 详细规则见：`.claude/references/writing-styles/fanqie_shuangwen.md`

## AI痕迹快速替换

**情感直白 → 动作/生理/决策**
- "他非常愤怒" → "他的拳头攥紧了，指节发白"
- "她很伤心" → "她没说话，只是把手机翻过去扣在桌上"
- "他感到震惊" → "他愣了三秒，才开口：'你说什么？'"

**元叙述 → 直接行动**
- "总而言之/可以说/综合以上" → 删除，直接写结论动作
- "值得注意的是" → 删除
- "首先/其次/最后" → 改为流畅叙述或短句

**句式单调 → 打破节奏**
- 连续三句同句式 → 至少改一处为短句爆点
- 长段落 → 插入单句段落制造停顿

**AI高频词替换清单**
| 替换掉 | 改为 |
|--------|------|
| 此外 | 删除或用"还有""另外" |
| 至关重要 | 直接说为什么重要 |
| 充满活力的 | 具体描述 |
| 深入探讨 | 删除 |
| 持久的/永恒的 | 具体说多久 |
| 格局（抽象） | 具体说什么情况 |
| 织锦（比喻） | 删除 |
| 宝贵的 | 删除或说具体价值 |
| 标志着 | 改为"是" |
| 见证了 | 改为"发生了" |

## 检查点（改写后验证）
| 检查项 | 标准 | 不通过处理 |
|--------|------|-----------|
| 开头冲突 | 前120字有冲突/风险/强情绪 | 重写开头 |
| 章末钩子 | 后80-150字有未闭合问题 | 重写结尾 |
| 长句比例 | >40字的句子 <10% | 拆分长句 |
| 解释段 | 无连续>200字的纯解释 | 打散或删除 |
| 句式多样 | 无连续3句相同结构 | 调整句式 |
| 情感具体 | 无“非常/十分+情绪词”直白表达 | 改为动作/生理反应 |

## 章长适配
- **标准章（3000-5000字）**: 完整执行所有规则
- **短章/过场章（<2000字）**:
  - 开头冲突可降为“前200字”
  - 局面变化频率可降为“每1500字一次”
  - 钩子强度可为 weak
