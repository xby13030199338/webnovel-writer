## v5.2 引入（v5.4 沿用）：基于审查报告修复

### 输入格式

```json
{
  "overall_score": 85,
  "issues": [
    {"agent": "ooc-checker", "type": "OOC", "severity": "high", "location": "第3段", "suggestion": "林天对敌人太客气，应更冷酷"},
    {"agent": "consistency-checker", "type": "POWER_CONFLICT", "severity": "critical", "location": "第5段", "suggestion": "筑基3层不能使用金丹期技能"}
  ],
  "pacing_analysis": {
    "quest_ratio": 0.4,
    "fire_ratio": 0.35,
    "constellation_ratio": 0.25
  },
  "pass": true

```

### 问题修复优先级

| 严重度 | 处理方式 |
|-------|---------|
| critical | **必须修复**，否则记录 deviation |
| high | 优先修复 |
| medium | 建议修复 |
| low | 可选修复 |

### 问题类型修复指南

| 问题类型 | 修复方法 |
|---------|---------|
| **OOC** (人物失真) | 调整角色言行，使其符合人设；若需要OOC，补充触发原因 |
| **POWER_CONFLICT** (战力冲突) | 修改能力描述，使用当前境界可用的技能/能力 |
| **TIMELINE_ISSUE** (时间线问题) | 调整时间描述，补充时间流逝过渡 |
| **LOCATION_ERROR** (地点跳跃) | 补充移动过程描述，或调整地点设定 |
| **PACING_IMBALANCE** (节奏失衡) | 调整 Strand 比例，增加缺失的内容类型 |
| **LOW_COOL_POINTS** (爽点不足) | 增加爽点密度，补充打脸/升级/收获等元素 |
| **CONTINUITY_BREAK** (连贯断裂) | 补充过渡句，连接断裂的场景 |

## 网文化规则（硬约束）

1. **开头 120 字必须出现冲突/风险/强情绪**
2. **每 800-1200 字至少一次局面变化**（转折、推进、意外、立场变化）
3. **结尾 80-150 字必须设置钩子**（未闭合问题/危险逼近/信息反转）
4. **对话每句必须带意图**（试探/威胁/诱导/掩饰）
5. **抽象判断句 → 动作/反应/代价**（把结论写成行为）

**章长适配**:
- 标准章（3000-5000字）：全部执行
- 短章/过场章（<2000字）：
  - 开头冲突可放宽至前 200 字
  - 局面变化频率可放宽为每 1500 字一次
  - 钩子强度允许为 weak

## AI 痕迹检测（Humanizer-zh 标准）

> 词频统计**仅作为提醒**，不作为硬性门槛。若明显超标，需修复并简要说明。
> 来源：Humanizer-zh（基于 Wikipedia:Signs of AI writing）

### 24种AI痕迹快速检查

**内容模式**
| 痕迹 | 警示词 | 修复方向 |
|------|--------|---------|
| 过度强调意义/遗产 | 标志着、见证了、至关重要、关键转折点、不可磨灭的印记 | 删掉意义宣告，改为具体事实 |
| 宣传广告式语言 | 充满活力的、令人叹为观止的、开创性的、迷人的 | 用具体细节替代形容词堆砌 |
| -ing结尾肤浅分析 | 突出/彰显……、反映/象征……、培养/促进…… | 删除句末现在分词虚假深度 |
| 模糊归因 | 专家认为、观察者指出、一些批评者认为 | 改为具体来源或直接删除 |
| 提纲式结尾 | 尽管……面临挑战、未来展望 | 改为具体事件或悬念 |

**语言语法模式**
| 痕迹 | 警示词 | 修复方向 |
|------|--------|---------|
| AI高频词 | 此外、至关重要、深入探讨、持久的、格局（抽象）、织锦、宝贵的 | 替换为普通词或删除 |
| 系动词回避 | 作为/代表/充当[一个]、拥有/设有[一个] | 改回"是" |
| 否定式排比 | 不仅……而且……、这不仅仅是……而是…… | 直接陈述核心意思 |
| 三段式过度 | 首先/其次/最后、第一/第二/第三 | 改为两项或流畅叙述 |
| 同义词循环 | 同一人物用主人公/主角/英雄/中心人物轮换 | 统一用同一称谓 |
| 虚假范围 | 从X到Y（但无意义尺度） | 直接列举具体内容 |

**风格模式**
| 痕迹 | 表现 | 修复方向 |
|------|------|---------|
| 破折号过度 | 每段多个破折号 | 改为逗号/句号，每段最多一个 |
| 粗体过度 | 正文大量加粗 | 删除正文粗体 |
| 内联标题列表 | **用户体验：** 用户体验通过…… | 改为流畅段落 |
| 表情符号 | 🚀✨💡 | 全部删除 |

**交流模式**
| 痕迹 | 警示词 | 修复方向 |
|------|--------|---------|
| 协作交流痕迹 | 希望这对您有帮助、当然！、请告诉我 | 全部删除 |
| 填充短语 | 值得注意的是、为了实现这一目标、在这个时间点 | 删除或精简 |
| 过度限定 | 可以潜在地、可能被认为、某种程度上可能 | 删掉多余限定词 |
| 通用积极结论 | 未来看起来光明、激动人心的时代即将到来 | 改为具体下一步或悬念 |

### 高频词提醒清单（完整版）

| 类型 | 关键词 |
|------|--------|
| 总结词 | 综合 / 总之 / 由此可见 / 总而言之 / 综上所述 |
| 列举结构 | 首先 / 其次 / 最后 / 第一 / 第二 / 第三 |
| 学术词 | 而言 / 某种程度上 / 本质上 / 深入探讨 |
| 因果连词 | 因为 / 所以 / 由于 / 因此（连续出现时） |
| 情绪直白 | 非常愤怒 / 十分高兴 / 极度悲伤 / 感到震惊 |
| AI高频词 | 此外 / 至关重要 / 充满活力 / 持久的 / 宝贵的 / 格局 |
| 意义宣告 | 标志着 / 见证了 / 体现了 / 象征着 / 证明了 |

### 去 AI 味修复手法

**句式打破**
- 连续3句同结构 → 至少改1句为短句或碎片化短语
- 长段落 → 插入单句段落制造停顿
- 顺序叙述 → 尝试逆向逻辑或省略主语

**情感具体化**
- "他非常愤怒" → "他的拳头攥紧了，指节发白"
- "她很伤心" → "她没说话，只是把手机翻过去扣在桌上"
- "他感到震惊" → "他愣了三秒，才开口：'你说什么？'"

**对话口语化**
- 加入犹豫："等等……这不可能对吧……"
- 加入语气词："你是认真的吧？"
- 用俚语/昵称替代正式称谓
- 通过粗犷对话加剧冲突，而非平和陈述

**信息密度降低**
- 每段聚焦一个重点
- 背景信息通过简短对话暗示，不做长篇解释
- 偶尔加入质疑或反讽打断叙述节奏

### 快速检查清单（Humanizer-zh）

- [ ] 连续三个句子长度相同？打断其中一个
- [ ] 段落以简洁的单行结尾？变换结尾方式
- [ ] 揭示前有破折号？删除它
- [ ] 解释了隐喻或比喻？相信读者能理解
- [ ] 使用了"此外""然而"等连接词？考虑删除
- [ ] 三段式列举？改为两项或四项

### 自然化质量评分（总分50，建议参考）

| 维度 | 评估标准 | 满分 |
|------|---------|------|
| 直接性 | 直接陈述事实还是绕圈宣告？ | 10 |
| 节奏 | 句子长度是否变化？长短交错 | 10 |
| 信任度 | 是否尊重读者智慧？简洁明了 | 10 |
| 真实性 | 听起来像真人说话吗？自然流畅 | 10 |
| 精炼度 | 还有可删减的内容吗？无冗余 | 10 |

> 45-50分优秀 / 35-44分良好 / 低于35分需重新修订

## 润色红线

- 改变情节走向 → 违反“**大纲即法律**”
- 修改主角实力（除非修复 POWER_CONFLICT）→ 违反“**设定即物理**”
- 改变人物关系 → 违反设定
- 删除伏笔 → 破坏长线剧情

---

## 风格一致性检查

### 视角一致
- 第一人称：全程“我”视角
- 第三人称限制：只描写主角能感知的
- 第三人称全知：可以切换视角

### 时态一致
- 过去时为主
- 避免时态混乱

### 文风一致
- 轻松吐槽风 vs 正剧严肃风
- 热血燃向 vs 冷静克制
- 保持全文统一
